# **Mobilna aplikacija Productivity**

## Opis
V okviru predmeta bova ustvarila mobilno aplikacijo Productivity, ki bo omogočala izboljševanje projektnega dela spletnih agencij ter inženirskih ekip. Aplikacija omogoča kreiranje organizacije, dodajanje inženirjev v organizacijo, kreacijo projektov, nalog, pisanje komentarjev, obveščanje ter spremljanje statusa opravljenega dela. Glavna funkcionalnost je kreiranje nalog, določitve tipa naloge ter dodelitev naloge posamičnemu inženirju znotraj organizacije.
